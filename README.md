# Course Withdrawal Application

**Course Withdrawal Application** is a desktop-based system that simplifies the process of withdrawing from courses. Students can submit withdrawal requests, track their status, and access related policies, while administrators can review, approve, or reject requests efficiently.

## Features

### Student Features
- View enrolled and available courses  
- Submit withdrawal requests with reasons  
- Track status of submitted requests  
- View course withdrawal policies and deadlines  

### Administrator Features
- View students and their course enrollments  
- Review, approve, or reject withdrawal requests  
- Edit course withdrawal policies and deadlines  

## Purpose
This application provides a user-friendly, structured process for managing course withdrawals, ensuring both students and administrators have clear and efficient workflows.

## Technologies Used
- **NetBeans IDE** – development environment  
- **Java Swing** – graphical user interface (GUI)  
- **MySQL** – database for storing courses and withdrawal requests  

## Scope
- Focused on the course withdrawal process  
- Does not integrate with course registration or grading systems  
- Data comes from predefined MySQL tables  

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/ridaaayaz/CourseWithdrawalApp.git
```
2. Open the project in NetBeans IDE.
3. Ensure MySQL is installed and create the database using the provided SQL scripts.
4. Update the database connection settings in the project.
5. Run the application
